Privacy Policy

ClipSpin for macOS — Last updated: 13 March 2026

ClipSpin is designed to be completely private. Your data never leaves your Mac. There is no network access, no analytics, no tracking, and no cloud sync.

Overview

ClipSpin is a clipboard manager that runs locally on your Mac. It monitors your system clipboard to provide clipboard history, pinning, and snippet features. This privacy policy explains what data ClipSpin accesses, how it is stored, and what it does not do.

Data ClipSpin Accesses

ClipSpin reads the contents of your macOS system clipboard in order to provide its core functionality. The types of data it may read from your clipboard include:

  • Text — plain text and URLs you copy
  • Images — screenshots and copied images (up to 2 MB)
  • Files and folders — file paths and associated icons when you copy items from Finder

ClipSpin also stores snippets you create manually, including any text, email addresses, phone numbers, addresses, URLs, or other information you choose to save.

How Your Data Is Stored

All data is stored locally on your Mac in the following location:

~/Library/Application Support/ClipSpin/

Data is saved as JSON files on disk. There is no database server, no encryption at rest, and no external storage. The data is accessible only to your macOS user account and to applications with appropriate file system permissions.

Data type Storage Retention
Clipboard history Local JSON file Up to 1,000 items; cleared manually
Pinned items Local JSON file Up to 5 items; removed manually
Snippets Local JSON file Unlimited; deleted manually
Preferences UserDefaults (macOS) Until app is removed

What ClipSpin Does Not Do

  • No network access — ClipSpin makes zero network requests. It does not connect to the internet for any reason.
  • No analytics or telemetry — ClipSpin does not collect usage data, crash reports, or any form of analytics.
  • No cloud sync — Your clipboard history and snippets are never uploaded to any server or cloud service.
  • No third-party SDKs — ClipSpin does not include any third-party analytics, advertising, or tracking frameworks.
  • No data sharing — Your data is never shared with, sold to, or accessible by any third party.
  • No user accounts — ClipSpin does not require registration, login, or any form of account creation.

Permissions

ClipSpin requests the following macOS permission:

  • Accessibility — Required to simulate keyboard events (Cmd+V) for the auto-paste feature. This permission is managed through macOS System Settings under Privacy & Security → Accessibility. You can revoke it at any time.

ClipSpin does not request access to your contacts, location, camera, microphone, or any other sensitive system resource.

Clipboard Monitoring

ClipSpin periodically checks the system clipboard (NSPasteboard) for changes. When a change is detected, the new content is saved to your local clipboard history. This monitoring runs only while ClipSpin is active in the menu bar. Quitting ClipSpin stops all clipboard monitoring immediately.

ClipSpin does not monitor clipboard activity from other apps in any way beyond reading the standard system pasteboard that macOS makes available to all applications.

Secure Input Fields

When a secure input field is active (such as a password field in your browser), macOS enables Secure Event Input mode. ClipSpin detects this and will not attempt to simulate keystrokes. ClipSpin relies on the standard macOS pasteboard APIs. Behaviour in secure input fields is controlled entirely by macOS.

Data Deletion

You have full control over your data at all times:

  • Clear history — Use the Clear button in Settings to delete all clipboard history.
  • Delete individual items — Right-click any item and choose Delete.
  • Delete snippets — Right-click any snippet and choose Delete.
  • Complete removal — Uninstalling ClipSpin and deleting the folder at ~/Library/Application Support/ClipSpin/ removes all stored data permanently.

Children's Privacy

ClipSpin does not knowingly collect any personal information from children. The app does not collect personal information from any user, regardless of age.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated revision date. Since ClipSpin has no network access, it cannot notify you of policy changes within the app.
